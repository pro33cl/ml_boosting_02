# ml_boosting_02
Notebook que construye y evalúa modelos Boosting para predecir enfermedades utilizando AdaBoost, GradientBoosting y XGBoost. Incluye carga y análisis de tres bases de datos, preprocesamiento, división train/test, tuning de hiperparámetros con GridSearchCV y evaluación mediante ROC, AUC y métricas de clasificación.
